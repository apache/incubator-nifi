Accessing Google Drive from NiFi

This processor uses Google Cloud credentials for authentication to access Google Drive. The following steps are required to prepare the Google Cloud and Google Drive accounts for the processors:

  1. Enable Google Drive API in Google Cloud - Follow instructions at https://developers.google.com/workspace/guides/enable-apis and search for 'Google Drive API'.
  2. Grant access to Google Drive files - In Google Cloud Console navigate to IAM & Admin -> Service Accounts. Take a note of the email of the service account you are going to use. Navigate to the parent of the folder of the files to be fetched in Google Drive. Right-click on the Folder -> Share. Enter the service account email.

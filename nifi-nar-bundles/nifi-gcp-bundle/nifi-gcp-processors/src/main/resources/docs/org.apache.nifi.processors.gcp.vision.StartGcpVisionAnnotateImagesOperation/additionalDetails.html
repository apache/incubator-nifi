Google Cloud Vision - Start Annotate Images Status

Usage

StartGcpVisionAnnotateImagedOperationStatus is designed to periodically check image annotation status. This processor should be used in pair with GetGcpVisionAnnotateImagesOperationStatus Processor. FlowFile will contain the serialized response that contains the result and additional metadata as it is documented in Google Vision API Reference.

JSON payload template - note that it can be simplified with the optional fields, check Vision API reference for more details - example: 

{
	"requests": [{
		"image": {
			"source": {
				"imageUri": "gs://..."
			}
		},
		"features": [{
			"type": "DOCUMENT_TEXT_DETECTION",
			"maxResults": 4
		}]
	}],
	"outputConfig": {
		"gcsDestination": {
			"uri": "gs://..."
		},
		"batchSize": 2
	}
}
